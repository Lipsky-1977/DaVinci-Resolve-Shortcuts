# DaVinci Resolve 19+ 快捷键查询

一个纯 HTML 快捷键查询工具，**无需安装任何依赖**，手机和 PC 浏览器均可直接打开使用。

## 使用方式

直接打开 `index.html` 即可，或将文件传到手机浏览器中。

## 功能

- **149 条快捷键**，覆盖 10 个常用分类
- 适用于 DaVinci Resolve 18 / 19 及更高版本
- **实时搜索**：按快捷键名称、按键组合或功能描述筛选
- **分类筛选**：播放控制、时间线、编辑、修剪、调色、Fusion、音频、页面切换、交付等
- **深色模式**：跟随系统主题自动切换
- **触控优化**：移动端单手操作适配
- **离线可用**：单文件，不依赖网络

## 快捷键分类

| 分类 | 说明 |
|------|------|
| 播放控制 | Space播放、JKL变速、I/O入出点 |
| 时间线导航 | 帧级导航、编辑点跳转、缩放 |
| 编辑与剪辑 | 撤销、复制、切割(B)、标记(M) |
| 切割与修剪 | A选择、T修剪、B切割、R卷动 |
| 调色 (Color) | 调色页面、Lift/Gamma/Gain、示波器 |
| Fusion合成 | 节点工具、查看器、渲染 |
| Fairlight音频 | 音频页面、自动模式、音量 |
| 页面切换 | Shift+1~9 切换工作区 |
| 交付 (Deliver) | 渲染队列、导出设置 |
| 文件/工具/面板 | F5-F10功能区、Ctrl+1-4面板 |
